# 💻 MIR MTZ 💻

## 📚 Descripción 📚
Aplicación móvil Android que ayuda en la generación de Matrices de Indicadores para Resultados.

## 🚀 Caracteristicas 🚀
- Inicio de sesión
- Guardado de datos necesarios de una MIR
- Generación de matrices en formato PDF

## 🚀 Tecnologías 🚀
- Java
- PHP
- MySQL
- Android
- Volley
